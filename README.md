# INLOW

INLOW是一个运行在x86上的类UNIX玩具

### PURPOSE
当前分支有非常详细的开发博客+必要注释，目的是让更多对OS感兴趣的人有更容易的学习途径。

### BUILD
安装必要工具并构建交叉编译环境可以参考我这篇博客[GCC-Cross-Compiler](https://blog.inlow.online/2019/01/04/GCC-Cross-Compiler/)

安装好交叉编译器后在项目目录执行`make`

### RUN
进入项目目录，执行`make qemu`
